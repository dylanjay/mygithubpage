Virtual Pets

Description

Web based Megaman Battle Network clone using Phaser, a WebGL game engine framework

Details

  • Arena where web based RPG pets can battle for rewards
  • Lead Gameplay Programmer alongside another team member
  • Used Phaser to handle sprite inclusion as well as a gameplay loop
  • Used Peer.js to handle peer to peer connections
  • Mainly worked on gameplay including game classes, sprite handling, and event handling.
