Racquetball (C#)

Description

Unity racquetball game made to work with Virtual Reality HMD's such as the Oculus Rift

Details

  • Used the Unity game engine to construct a full racquetball court
  • Racquetball game made to play in the third person using actual racquetball rule sets
  • Mainly worked on collision detection and gameplay physics using ray casting and unity physics
